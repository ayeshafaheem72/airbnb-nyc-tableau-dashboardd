Project Description

This project focuses on designing an interactive data visualization dashboard that explores Airbnb listings in New York City. The goal of the dashboard is to transform a large and complex dataset into meaningful visual insights that help users understand pricing trends, neighborhood patterns, booking seasonality, and host activity.

Instead of relying on static charts, this project emphasizes interactivity and temporal analysis, allowing users to explore the data dynamically through filters, maps, and time-based visualizations. The dashboard is designed for both analytical clarity and ease of exploration.

Why This Dataset?

As students, we often work with datasets that feel abstract or disconnected from real-world impact. We chose the NYC Airbnb dataset because it represents a system most people are already familiar with: short-term rentals, pricing competition, and urban living.

The dataset contains thousands of listings across diverse neighborhoods, making it ideal for practicing exploratory data analysis, spatial visualization, and temporal trend analysis. Its real-world relevance helped keep the analysis engaging rather than feeling like another repetitive academic exercise.

Dataset Description

The dataset used in this project is AB_NYC_2019.csv, sourced from the Inside Airbnb initiative. It includes information about Airbnb listings across New York City.

  • Approximately 49,000 listings
  • 221 unique neighborhoods
  • 5 neighborhood groups (Manhattan, Brooklyn, Queens, Bronx, Staten Island)
  • Attributes include price, room type, number of reviews, availability, and host information

The dataset supports both numerical and categorical analysis, making it well-suited for building a multi-layered interactive dashboard.

Tools and Technologies Used

  • Tableau – for building interactive and temporal visualizations
  • GitHub Pages – for hosting the project webpage
  • HTML & CSS – for structuring and styling the project website

Tableau was selected because it allows rapid development of professional dashboards while supporting advanced features such as filters, maps, calculated fields, and dynamic interaction without extensive programming overhead.
