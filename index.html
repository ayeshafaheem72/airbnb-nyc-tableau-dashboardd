Project Description

This project presents an interactive data visualization dashboard designed to analyze Airbnb listings across New York City. The objective is to transform a large real-world dataset into meaningful insights related to pricing trends, neighborhood distributions, seasonal demand, and host activity.

The dashboard emphasizes interactivity and temporal analysis, enabling users to dynamically explore patterns through filters, maps, and time-based visualizations rather than static charts.

Why This Dataset?

The NYC Airbnb dataset was chosen because it represents a real-world system that directly impacts urban housing, tourism, and pricing dynamics. Unlike abstract datasets, this data is familiar and relatable, making the analysis more engaging and meaningful.

Its scale and diversity make it ideal for exploratory data analysis, spatial visualization, and identifying temporal trends.

Dataset Description

The dataset used in this project is AB_NYC_2019.csv, sourced from the Inside Airbnb initiative.

  • Approximately 49,000 Airbnb listings
  • 221 unique neighborhoods
  • 5 neighborhood groups (Manhattan, Brooklyn, Queens, Bronx, Staten Island)
  • Attributes include price, room type, number of reviews, availability, and host details

Tools and Technologies Used

  • Tableau – interactive dashboard development
  • GitHub Pages – project hosting
  • HTML & CSS – webpage structure and styling

Tableau was selected for its strong support of interactive filtering, geographic visualizations, and rapid dashboard prototyping without extensive programming overhead.
