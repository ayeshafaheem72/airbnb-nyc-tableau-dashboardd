NYC Airbnb Interactive Dashboard

Team Members

Ayesha Faheem

Project Description

This project presents an interactive Tableau dashboard analyzing Airbnb listings in New York City. The dashboard explores pricing trends, neighborhood distributions, booking seasonality, and host activity using temporal and interactive visualizations.

Dataset Description

Dataset: AB_NYC_2019.csv
Source: Inside Airbnb
Size: ~49,000 listings across 221 NYC neighborhoods

Tools & Technologies

  • Tableau
  • GitHub Pages
  • HTML & CSS
