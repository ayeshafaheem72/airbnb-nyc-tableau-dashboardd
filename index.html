Project Description

This project presents an interactive data visualization dashboard designed to explore Airbnb listings across New York City. The objective is to transform a large real-world dataset into meaningful insights related to pricing trends, neighborhood distributions, seasonal demand, and host activity.

The dashboard emphasizes interactivity and temporal analysis, allowing users to dynamically explore patterns through filters, maps, and time-based visualizations rather than static charts.

Why This Dataset?

We selected the NYC Airbnb dataset because it represents a real system that directly affects urban life, housing, and tourism. Unlike abstract datasets often used in coursework, this data reflects real economic and geographic patterns that users can easily relate to.

Its size and diversity make it ideal for practicing exploratory data analysis, spatial mapping, and temporal trend identification.

Dataset Description

The dataset used is AB_NYC_2019.csv, sourced from the Inside Airbnb initiative.

  • ~49,000 Airbnb listings
  • 221 unique neighborhoods
  • 5 neighborhood groups
  • Attributes include price, room type, number of reviews, availability, and host details

Tools and Technologies Used

  • Tableau – interactive dashboard development
  • GitHub Pages – project hosting
  • HTML & CSS – webpage structure and styling

Tableau was chosen for its strong support of interactive filtering, geographic visualizations, and rapid dashboard prototyping without requiring extensive programming.
